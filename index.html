SCOUT Project

Exploring Environment Mapping and Autonomous Movement

About

This repository contains scripts and data for SCOUT, an autonomous robot that maps its environment using a ToF camera and odometry. Use the links below to explore the project files and outputs.

  • Sensor Data (CSV)

Scripts

  • Map Processor Script
  • Movement Logger Script

How to Run

Follow these steps to use the project:

  1. Clone this repository to your local machine.
  2. Ensure you have the necessary Python libraries installed.
  3. Run the scripts in the scripts/ folder as needed.
  4. Check the data/ folder for outputs.
